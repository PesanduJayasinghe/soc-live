# Workspace Instructions

Welcome to the **Soc Ops** development workspace!

## 🚀 Quick Start

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the development server**
   ```bash
   npm run dev
   ```
   The app will be available at the local address shown in your terminal.

3. **Lint, Build, and Test**
   - Lint: `npm run lint`
   - Build: `npm run build`
   - Test: `npm run test`

## 🛠️ Development Notes

- Uses **React** + **Vite** + **Tailwind CSS v4** (see .github/instructions/tailwind-4.instructions.md for advanced usage).
- For frontend design, see .github/instructions/frontend-design.instructions.md for creative guidelines.
- All source code is in the `src/` directory.
- Workshop/lab guides are in the `workshop/` folder.

## 🧑‍💻 Onboarding

- Node.js 22+ required.
- For Codespaces/Dev Containers, open in container for a pre-configured environment.
- See [workshop/01-setup.md](workshop/01-setup.md) for a step-by-step setup flow.
